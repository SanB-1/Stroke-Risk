Our predictions are based on real anonymous data shared by patients like you, in this section we explain how the data was analyzed to create this tool. Some technical knowledge might be needed to understand this section.


The relationship between age and stroke ocurrence is visualized with a bar plot and a box plot. Strokes appear to be more prevalent in older patients.


On this heatmap, we can see the correlation between different features and stroke risk. Only binary (0 and 1 values) and numerical features are used for this graph. Features with a value closer to one have a higher correlation with stroke risk as they get higher, features with a value closer to -1 have a lower correlation with stroke risk as they get higher. Values close to 0 either have no correlation with stroke risk or have a non-linear distribution.


The following graph is called a confusion matrix, it allows us to measure the accuracy of the machine learning model used to predict your risk of stroke. We can see that our software showed no false negatives and only 13 false positives when tested.
